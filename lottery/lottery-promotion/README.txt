                                彩票活动模块

说明
=================================        
周期性的执行活动相关处理过程。
包括的活动：
    1、竞彩足球奖上加奖活动
    2、注册充值送10元活动
    3、竞彩篮球奖上加奖活动
    
启动、停止
=================================
    启动：./run.sh start
    停止：./run.sh stop

编译打包
=================================
生成测试环境部署包：
    mvn clean package

生成正式环境部署包：
    mvn clean package -P production
